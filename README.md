# 💻 InfoPCJS — Instant System Insights

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Vite](https://img.shields.io/badge/built%20with-Vite-646CFF?logo=vite)](https://vitejs.dev/)
[![JavaScript](https://img.shields.io/badge/Language-JavaScript-F7DF1E?logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

**InfoPCJS** is a lightweight, high-performance web application designed to fetch and display hardware and system specifications directly within the browser. Built with a focus on speed and privacy, it provides a clean interface for developers and power users to audit their environment.

🌐 **Live Demo:** [https://infopcjs.netlify.app/](https://infopcjs.netlify.app/)

---

## 🚀 Key Features

- **Real-Time Data:** Instantly retrieves hardware concurrency, browser engine details, and OS architecture.
- **Minimalist UI:** A clean, responsive dashboard designed for clarity and ease of use across all devices.
- **Privacy First:** All data processing happens on the client side. No system information is ever sent to a server.
- **Vite-Powered:** Optimized for near-instant load times and a smooth user experience.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
- **Logic:** JavaScript (ES6+)
- **Bundler:** [Vite](https://vitejs.dev/)
- **Deployment:** [Netlify](https://www.netlify.com/)
